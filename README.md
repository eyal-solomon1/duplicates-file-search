# empty-file-search
a small tool for searching duplicate files

#### how to use :
`go run <directory_path>`

### example output :
```
a299b04 2
   .git/logs/HEAD
   .git/logs/refs/heads/main
e8191b3 2
   same-files/file1.log
   same-files/file2.log
a3a488c 2
   .git/refs/heads/main
   .git/refs/remotes/origin/main
```
